Flexbox Row

Use these three properties to create a Flexbox row layout

function  () {
                return ".row {
                display: flex;
                flex-direction: row;
                flex-wrap: wrap;
                }"    
                }

Flexbox Column

function  () {
                return ".column {
                        display: flex;
                        flex-direction: column;
                    }"    
                }

Flexbox Grid

function () {
            return ".grid {
                display: grid;
            }"
                
        }
